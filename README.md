## Raz React UI kit
Ui components for react  
Forked from [tayeh-react](https://github.com/PchasSoftware/tayeh-react)

#### Goals
- simplicity 
- precision 

#### Usage 
Installing via npm
```
	npm install --save raz-ui
``` 
  
Installing via pnpm
```
	pnpm add raz-ui
```   

Installing via pnpm
```
	yarn add raz-ui
```
